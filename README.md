# semicolonproject2
[[ 시간표 연동 지도 프로젝트 ]]

기획의도 : 학과 신입생을 위한 시간표 연동 프로젝트 생성

기간 : 2021.10.01 ~ 2021.12.20

개발 인원 : 5명

개발 플랫폼 : Window8

개발 툴 : Eclipse

사용언어 : JAVA, HTML, CSS, JavaScript

사용기술 : JQuery, Vworld(Vworld API), KAKAO API(2021)

담당 업무 : 지도 및 로드뷰 API 

기능 : 시간표 작성가능, 지도 확인가능, 로드뷰 확인가능, VWorldAPI를 활용하여 2D 지도 검색 API를 연동 후 마커 표시 기능
